# Actividades de Cultura Digital

Este repositorio contiene un script de Python para el procesamiento de datos básicos.

## Descripción del código
El archivo `analisis.py` tiene la función de **calcular el promedio aritmético** de una lista de valores numéricos. 

### ¿Cómo funciona?
1. Define una función llamada `calcular_promedio`.
2. Toma una lista de números como entrada.
3. Suma todos los valores y los divide por la cantidad total de elementos.
4. Imprime el resultado final en la consola.

Este es un ejemplo de automatización de tareas matemáticas simples mediante programación.
